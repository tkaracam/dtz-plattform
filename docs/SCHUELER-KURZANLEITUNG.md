# DTZ-LIB Schüler-Kurzanleitung

## 1. Anmeldung
1. `index.html` öffnen.
2. Mit Benutzername und Passwort anmelden.

## 2. Module im Menü
- `Portal`
- `Saal`
- `Simulation`
- `Schreiben`
- `Sprechen`
- `LiD`
- `BSK`

## 3. Virtueller Saal
1. `Saal` öffnen.
2. Raumcode eingeben.
3. `Raum beitreten` klicken.
4. Aufgabenstellung lesen.
5. Antwort im Zeitfenster schreiben.
6. `Antwort abgeben` klicken.

## 4. Prüfungssimulation (Schreiben)
1. `Simulation` öffnen.
2. Prüfungszeit auswählen.
3. `Simulation beginnen` klicken.
4. Antwort schreiben.
5. Manuell abgeben oder automatische Sperre bei Zeitende.

## 5. Schreiben-Korrektur
1. `Schreiben` öffnen.
2. Text eingeben.
3. `Brief korrigieren` klicken.
4. Optional: `Brief hochladen`.

## 6. Portal nutzen
Im `Portal` finden Sie:
- `Meine Ergebnisse`
- `Meine Hausaufgaben`
- `Lehrkraft-Notizen`
- `Zertifikatsvorbereitung` (DTZ/DTB)

## 7. Hinweise
- Zugangsdaten nicht weitergeben.
- Aufgaben im vorgegebenen Zeitrahmen bearbeiten.
- Bei Login-Problemen Lehrkraft kontaktieren.
