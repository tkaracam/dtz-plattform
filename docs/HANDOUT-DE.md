# DTZ-LIB Handout (Lehrkraft + Schüler)

## Für Lehrkräfte
- Anmeldung über `admin.html`
- Schülerkonten erstellen
- Virtuellen Saal starten
- Live überwachen
- Bewerten (einzeln/gesamt)
- CSV exportieren

## Für Lernende
- Anmeldung über `index.html`
- Saal mit Code beitreten
- Simulation schreiben
- Ergebnisse im Portal prüfen

## Dokumente
- Lehrkraft: `docs/LEHRKRAFT-KURZANLEITUNG.md`
- Schüler: `docs/SCHUELER-KURZANLEITUNG.md`
