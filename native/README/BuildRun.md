# DTZ-LID edu – Build & Run

## Android Studio
1. Open project: `native/android/DTZLiDedu`
2. Wait for Gradle sync
3. Select device/emulator
4. Run ▶

## Xcode
1. Open project: `native/ios/DTZLiDedu.xcodeproj`
2. Set Team & Bundle ID
3. Select simulator/device
4. Run ▶

## Notes
- Android uses Jetpack Compose + OkHttp.
- iOS uses SwiftUI + URLSession.
