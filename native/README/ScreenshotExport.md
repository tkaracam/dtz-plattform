# Screenshot Export Flow (Pragmatic)

## iOS (Xcode Simulator)
1. Open project: `native/ios/DTZLiDedu.xcodeproj`
2. Run on simulator (6.7", 6.5")
3. Use Simulator: File > New Screenshot
4. Save into `native/README/assets/screenshots/ios-*.png`

## Android (Android Studio Emulator)
1. Open project: `native/android/DTZLiDedu`
2. Run on emulator (1080x2400)
3. Use emulator camera icon to capture
4. Save into `native/README/assets/screenshots/android-*.png`
