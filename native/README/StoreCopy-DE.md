# Store Copy – Deutsch (Final Entwurf)

## App Store / Play Store
**Name:** DTZ-LID edu
**Kurzbeschreibung:** DTZ Training A2–B1: Hören, Lesen, Schreiben, Korrekturen.

**Lange Beschreibung:**
DTZ-LID edu ist eine Trainingsplattform für den Deutsch-Test für Zuwanderer (A2–B1).
Üben Sie Hören, Lesen und Schreiben in prüfungsnahen Aufgaben und erhalten Sie Korrekturen im Portal.

Funktionen:
- DTZ Hören Teil 1–4
- DTZ Lesen Teil 1–5
- Schreiben & Upload
- Korrigierte Briefe im Portal

Hinweis: Inhalte sind prüfungsnah, aber keine offiziellen Prüfungsunterlagen.

**Keywords:**
DTZ, Deutsch, A2, B1, Hören, Lesen, Schreiben, Prüfung, Training, Integrationskurs
