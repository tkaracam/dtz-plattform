# DTZ-LID edu – Screenshot Guide (Platzhalter)

## Ziel
Store-Screenshots, die die Kernfunktionen zeigen:
1) Login
2) DTZ Training (Hören/Lesen)
3) Schreiben (Brief hochladen)
4) Portal (Korrigierte Briefe)
5) Einstellungen/Abmelden

## iOS Größen
- 6.7" (1290x2796)
- 6.5" (1242x2688)
- iPad (optional)

## Android Größen
- Phone (1080x1920 oder 1080x2400)
- Tablet (optional)

- Kontrastreiche UI
