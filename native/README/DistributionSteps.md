# DTZ-LID edu – Distribution Steps (Placeholder)

## Android (Google Play)
1. Open `native/android/DTZLiDedu` in Android Studio.
2. Update versionCode / versionName in `app/build.gradle`.
3. Generate signed bundle: Build > Generate Signed Bundle/APK > Android App Bundle.
4. Upload AAB to Play Console.

## iOS (App Store)
1. Open `native/ios/DTZLiDedu.xcodeproj` in Xcode.
2. Set Team and Bundle Identifier.
3. Archive: Product > Archive.
4. Upload to App Store Connect.
