1. Bildung des Partizips II

VerbtypMusterBeispiel
Regelmäßigge- + Stamm + -tmachen → gemacht
Regelmäßig (-d/-t)ge- + Stamm + -etarbeiten → gearbeitet
Unregelmäßigge- + Stamm (verändert) + -enfahren → gefahren
TrennbarPräfix + ge- + Stamm + -t/-enaufmachen → aufgemacht
Untrennbar / -ierenKein ge-!verstehen → verstanden; studieren → studiert
