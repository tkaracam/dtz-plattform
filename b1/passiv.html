Das Passiv (B1)

Passiv Präsens, Passiv Präteritum und Passiv Perfekt – Bildung und Verwendung.

← B1-Themen

1. Passiv Präsens – wird + Partizip II

PersonBildungBeispiel
ichwerde + P.IIIch werde gefragt.
duwirst + P.IIDu wirst eingeladen.
er/sie/eswird + P.IIDer Motor wird geprüft.
wirwerden + P.IIWir werden informiert.
ihrwerdet + P.IIIhr werdet benachrichtigt.
sie/Siewerden + P.IIDie Pakete werden geliefert.
Im Passiv steht das Objekt des Aktivsatzes als Subjekt. Der Handelnde wird oft nicht genannt.
Aktiv: Der Arzt untersucht den Patienten.
Passiv: Der Patient wird untersucht.

2. Passiv Präteritum – wurde + Partizip II

PersonBildungBeispiel
ichwurde + P.IIIch wurde informiert.
duwurdest + P.IIDu wurdest gefragt.
er/sie/eswurde + P.IIDer Motor wurde geprüft.
wirwurden + P.IIWir wurden eingeladen.
ihrwurdet + P.IIIhr wurdet benachrichtigt.
sie/Siewurden + P.IIDie Briefe wurden geschrieben.
Aktiv: Man hat das Haus gebaut.
Passiv Präteritum: Das Haus wurde gebaut.

3. Passiv Perfekt – ist/sind + Partizip II + worden

PersonBildungBeispiel
ichbin + P.II + wordenIch bin gefragt worden.
dubist + P.II + wordenDu bist eingeladen worden.
er/sie/esist + P.II + wordenDer Motor ist geprüft worden.
wirsind + P.II + wordenWir sind informiert worden.
ihrseid + P.II + wordenIhr seid benachrichtigt worden.
sie/Siesind + P.II + wordenDie Pakete sind geliefert worden.
Im Passiv Perfekt steht worden (nicht geworden!).
Aktiv: Der Chef hat den Mitarbeiter entlassen.
Passiv Perfekt: Der Mitarbeiter ist entlassen worden.

4. Handelnde Person: von + Dativ / durch + Akkusativ

PräpositionVerwendungBeispiel
von + DativPerson oder konkreter VerursacherDas Fenster wurde von dem Sturm zerstört.
durch + AkkusativMittel, Ursache, InstrumentDie Stadt wurde durch ein Erdbeben zerstört.
Der Brief wurde von der Sekretärin geschrieben.
Die Brücke wurde durch den Fluss beschädigt.
