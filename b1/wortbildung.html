Wortbildung (B1)

Komposita, Ableitungen mit Suffixen und Präfixen, Verb- und Adjektivbildung.

← B1-Themen

1. Komposita (Zusammengesetzte Nomen)

BildungGenusBeispiel
Nomen + Nomen= letztes Wortder Zug + das Ticket = das Zugticket
Adjektiv + Nomen= letztes Wortgroß + die Stadt = die Großstadt
Verb + Nomen= letztes Wortschlafen + das Zimmer = das Schlafzimmer
Nomen + -s- + Nomen (Fugen-s)= letztes Wortder Arbeit + s + das Zimmer = das Arbeitszimmer
Das Genus des Kompositums = Genus des letzten Bestandteils.

2. Nomen-Bildung mit Suffixen

SuffixGenusBasis → Nomen
-ungdiewohnen → die Wohnung
-heitdiekrank → die Krankheit
-keitdiemöglich → die Möglichkeit
-schaftdieFreund → die Freundschaft
-nisdas/dieergebnis → das Ergebnis
-erderlehren → der Lehrer
-indieder Lehrer → die Lehrerin
-lingderlernen → der Lehrling

3. Adjektiv-Bildung mit Suffixen

SuffixBasis → AdjektivBedeutung
-lichfreund → freundlichEigenschaft
-igHunger → hungrigZustand/Eigenschaft
-ischKind → kindischwie … / typisch für
-losArbeit → arbeitslosohne
-vollSinn → sinnvollmit viel
-barmachen → machbarkann gemacht werden
-samaufmerksam → aufmerksamEigenschaft

4. Verbpräfixe – trennbar und untrennbar

PräfixTypBeispiel
an-, auf-, ab-, ein-, aus-, mit-, vor-, zu-trennbaranrufen: Ich rufe an.
be-, ge-, er-, ver-, zer-, ent-, emp-, miss-untrennbarbezahlen: Ich bezahle.
Trennbar: aufmachen → Ich mache die Tür auf.
Untrennbar: verstehen → Ich verstehe das. (kein ge- im Partizip)
